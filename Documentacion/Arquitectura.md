# Arquitectura del Proyecto - NetWize

## Descripción General

NetWize es una aplicación web orientada al entrenamiento y simulación de soporte técnico nivel 1 (N1). Su propósito es permitir a los usuarios practicar diagnósticos básicos mediante escenarios interactivos y consultar conceptos fundamentales de redes y soporte IT.

---

## Tecnologías Utilizadas

### Frontend

* React
* JavaScript
* HTML5
* CSS3

### Herramientas de Desarrollo

* Visual Studio Code
* Git
* GitHub

### Despliegue

* Vercel (planeado)

---

## Estructura General

La aplicación estará compuesta por tres módulos principales:

### Dashboard

Permitirá visualizar estadísticas, historial de simulaciones y progreso del usuario.

### Simulador

Permitirá resolver escenarios de soporte técnico mediante preguntas y respuestas guiadas.

### Enciclopedia N1

Permitirá consultar conceptos fundamentales relacionados con soporte técnico y redes.

---

## Navegación

La aplicación utilizará un menú lateral para acceder a los diferentes módulos:

* Dashboard
* Simulador
* Enciclopedia N1
