# Árboles de Decisión - NetWize

## Caso 1: No tengo Internet

Problema inicial:
El usuario indica que no tiene conexión a internet.

Flujo:

1. Verificar si el router está encendido.
2. Revisar estado de luces del router.
3. Verificar conexión del dispositivo a la red WiFi.
4. Revisar configuración IP.
5. Aplicar solución o escalar soporte.

---

## Caso 2: WiFi lento

Problema inicial:
El usuario tiene conexión pero presenta baja velocidad.

Flujo:

1. Validar si afecta todos los dispositivos.
2. Revisar distancia al router.
3. Verificar saturación de red.
4. Revisar configuración WiFi.
5. Aplicar solución.

---

## Caso 3: Problemas DNS

Problema inicial:
El usuario tiene conexión pero no puede acceder a páginas.

Flujo:

1. Validar conexión a internet.
2. Realizar prueba de conectividad.
3. Verificar resolución DNS.
4. Cambiar servidores DNS si es necesario.
5. Confirmar funcionamiento.
