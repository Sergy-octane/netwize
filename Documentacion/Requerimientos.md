# Requerimientos del Sistema - NetWize

## Requerimientos Funcionales

### RF-01

El sistema debe permitir acceder al Dashboard principal.

### RF-02

El sistema debe permitir acceder al módulo Simulador.

### RF-03

El sistema debe mostrar preguntas y respuestas interactivas para la resolución de casos de soporte técnico.

### RF-04

El sistema debe mostrar un diagnóstico o solución al finalizar una simulación.

### RF-05

El sistema debe permitir acceder a la Enciclopedia N1.

### RF-06

El sistema debe permitir consultar información sobre IP, DNS, DHCP, Ping y Tracert.

## Requerimientos No Funcionales

### RNF-01

La aplicación debe ser responsive para dispositivos móviles y computadores.

### RNF-02

La interfaz debe ser intuitiva y fácil de usar.

### RNF-03

La aplicación debe ejecutarse en navegadores modernos.

### RNF-04

El sistema debe mantener tiempos de carga inferiores a 3 segundos en condiciones normales.

### RNF-05

El código fuente debe mantenerse bajo control de versiones mediante Git y GitHub.
